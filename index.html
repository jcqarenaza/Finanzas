Google Sheets
Encontralo en la URL: docs.google.com/spreadsheets/d/[ID]/edit
Creá una API Key en Google Cloud Console
Instrucciones de Configuración
1. Crear el Google Sheet
Creá una hoja de cálculo nueva en Google Sheets. Copiá el ID de la URL.
2. Hacer pública la hoja
Archivo → Compartir → "Cualquiera con el vínculo" → Lector (o Editor si querés escritura).
3. Obtener API Key
En Google Cloud Console, habilitá la "Google Sheets API" y creá una API Key.
4. Estructura de hojas
La app va a crear automáticamente las hojas: Meses, Ingresos, Gastos, Inversiones, Cuotas
⚠ Nota importante
Con solo API Key podés LEER datos. Para ESCRIBIR necesitás OAuth2 o usar un Google Apps Script de intermediario (te puedo generar el script).
